使用 HTML模板编写 不用再执行JS,填上适当的参数即可自动生成。其中 mod-xxx 为 js方法中的参数，根据 "-"来区分单词
如 titleLeft 在html中为 mod-title-left 属性。

Javascript写法

//javascript
mod.button(target,{size:"small",type:"submit"})；

mod.button(target,op) 参数说明

属性 类型 说明
target {object|string} 需要执行模块化的对象
op {object} 参数设置
属性 类型 支持版本 说明
size {string} >= 1.0 按钮大小 small|big|不填
type {string} >= 1.0 按钮类型 submit|clear|1|2|3
蓝色 submit|1
绿色 2
白色 clear|3

mod.button(target,op) 返回:

属性 类型 说明
btn {object} 按钮对象
op {object} 参数设置
属性 类型 支持版本 说明
btn.enable() {function} >= 1.0 按钮变为 可点状态 此时 btn.disabled == false
btn.disable() {function} >= 1.0 按钮变为 不可点状态 此时 btn.disabled == false
btn.disabled {boolean} >= 1.0 按钮是否属于不可点状态
