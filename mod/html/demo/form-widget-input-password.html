使用 HTML模板编写 不用再执行JS,填上适当的参数即可自动生成。其中 mod-xxx 为 js方法中的参数，根据 "-"来区分单词
如 titleLeft 在html中为 mod-title-left 属性。

密码

HTML模板写法

<tr>
	<th>
		<label for="password">密码</label>
	</th>
	<td>
		<input id="password" name="password" type="password" mod-required="true" mod-format="password">
	</td>
</tr>
<tr>
	<th>
		<label for="password2">重复密码</label>
	</th>
	<td>
		<input id="password2" type="password" mod-required="true" mod-format="password2" mod-linked="#password" />
	</td>
</tr>

javascript写法

<!-- html -->
<table class="bs_table_s0">
	<tr>
		<th>
			<label for="passwords1">密码</label>
		</th>
		<td>
			<input id="passwords1" name="password" type="password">
		</td>
	</tr>
	<tr>
		<th>
			<label for="password2s1">重复密码</label>
		</th>
		<td>
			<input id="password2s1" type="password" />
		</td>
	</tr>
</table>		
	
//js
mod.widget("#passwords1",{
	required:true,
	format:"password"
});	
mod.widget("#password2s1",{
	required:true,
	format:"password2",
	linked:"#passwords1"
});

mod.widget(target,op) 参数说明

属性 类型 说明
target {object|string} 需要执行模块化的对象
op {object} 参数设置
属性 类型 支持版本 说明
format {string} >= 1.0 格式化类型
密码控件 password
确认密码控件 password2
required {boolean} >= 1.0 是否为必填项 默认 false
linked {string} >= 1.0 关联( password2用参数)
errorText {string} >= 1.0 设置校验内容时错误的提示语
defaultText {string} >= 1.0 设置控件默认的提示语
oncheck(value) {function} >= 1.0 表单校验
@param value {object} 文本框内容
@return data {object} 如果返回 true 或者 undefined 则表示成功，返回其他则当做是错误提示内容
