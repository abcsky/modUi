HTML模板写法

使用 HTML模板编写 不用再执行JS,填上适当的参数即可自动生成。其中 mod-xxx 为 js方法中的参数，根据 "-"来区分单词
如 titleLeft 在html中为 mod-title-left 属性。

HTML模板写法

纯css样式控制，理论上无限极菜单

<div class="bs_menu" mod-show="true" mod-checkbox="true">
	<ul>
		<li><a mod-value="1">一级标题</a></li>
		<li>
			<a mod-value="">一级标题</a>
			<ul>
				<li><a mod-value="2-1">二级标题</a></li>
				<li><a mod-value="2-2">二级标题</a></li>
			</ul>
		</li>
		<li>
			<a mod-value="">一级标题</a>
			<ul>
				<li><a mod-value="3-1">二级标题</a></li>
				<li>
					<a mod-value="">二级标题</a>
					<ul>
						<li><a mod-value="3-2-1">三级标题</a></li>
						<li><a mod-value="3-2-2" mod-checked="true">三级标题</a></li>
					</ul>
				</li>
			</ul>
		</li>
	</ul>
</div>

HTML模板写法

纯css样式控制，理论上无限极菜单

<div class="bs_menu"  mod-show="true"  mod-href-target="_blank" >
	<ul>
		<li><a href="#">一级标题</a></li>
		<li>
			<a href="#">一级标题</a>
			<ul>
				<li><a href="#">二级标题</a></li>
				<li><a href="#">二级标题</a></li>
			</ul>
		</li>
		<li>
			<a href="#">一级标题</a>
			<ul>
				<li><a href="#">二级标题</a></li>
				<li>
					<a href="#">二级标题</a>
					<ul>
						<li><a href="#">三级标题</a></li>
						<li><a href="#">三级标题</a></li>
					</ul>
				</li>
			</ul>
		</li>
	</ul>
</div>

mod.menu(target,op) 参数说明

属性 类型 说明
target {object|string} 需要执行模块化的对象
op {object} 参数设置
属性 类型 支持版本 说明
checkbox {boolean} >= 1.20 显示checkbox，默认为false
icon {boolean} >= 1.20 显示图标。默认为 true
hrefTarget {string} >= 1.20 链接打开方式。默认为不填
show {boolean} >= 1.20 是否默认展示全部。默认为 false

mod.menu(type,op) 返回：

属性 类型 说明
menu {object} 菜单对象
属性 类型 支持版本 说明
menu.get() {function} >= 1.20 获取数据, checkbox 存在时适用方法
属性 类型 支持版本 说明
type {string} >= 1.20 "checked" - 获取选中的数据
"unchecked" - 获取没选中的数据
不填 - 获取所有数据
pop.current(href) {function} >= 1.20 设定选中的菜单。多级菜单适用方法
属性 类型 支持版本 说明
href {string} >= 1.20 链接地址
