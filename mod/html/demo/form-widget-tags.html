使用 HTML模板编写 不用再执行JS,填上适当的参数即可自动生成。其中 mod-xxx 为 js方法中的参数，根据 "-"来区分单词
如 titleLeft 在html中为 mod-title-left 属性。

词条控件-通过输入来设置词条

HTML模板写法

<!-- html -->
<form class="bs_form">
	<table class="bs_table_s0">
		
		<tr>
			<th>
				<label for="tags">标签设置</label>
			</th>
			<td>
				<input id="tags" name="tags" mod-width="500" type="hidden" mod-required="true" mod-format="tags" mod-split="," mod-type="edit">
			</td>
		</tr>
	</table>
</form>

javascript 写法

<!-- html -->
<table class="bs_table_s0">
	<tr>
		<th>
			<label for="tags1">标签设置</label>
		</th>
		<td>
			<input id="tags1" name="tags" type="hidden">
		</td>
	</tr>
</table>
	
//js
mod.widget("#tags1",{
	format:"tags",
	required:true,
	split:",",
	width:500,
	type:"edit"
});

词条控件-通过选择来设置词条

HTML模板写法

<!-- html -->
<form class="bs_form">
	<table class="bs_table_s0">
		
		<tr>
			<th>
				<label for="tags2">标签设置</label>
			</th>
			<td>
				<input id="tags2" name="tags" mod-width="500" type="hidden" mod-required="true" mod-format="tags" mod-split="," mod-type="select" mod-onselect="$('#tags2')[0].add('hello kugou' + Math.random())">
			</td>
		</tr>
	</table>
</form>

javascript 写法

<!-- html -->
<table class="bs_table_s0">
	<tr>
		<th>
			<label for="tags3">标签设置</label>
		</th>
		<td>
			<input id="tags3" name="tags" type="hidden">
		</td>
	</tr>
</table>
	
//js
mod.widget("#tags3",{
	format:"tags",
	required:true,
	split:",",
	width:500,
	type:"select",
	onselect:function(){
		mod.dialog("selectTag",{
			title:"标签选择",
			content:[
				'<a class="right10" onclick="$(\'#tags3\')[0].add(this.innerHTML)">标签01</a>',
				'<a class="right10" onclick="$(\'#tags3\')[0].add(this.innerHTML)">标签02</a>',
				'<a class="right10" onclick="$(\'#tags3\')[0].add(this.innerHTML)">标签03</a>'
			].join("")
		});
	}
});

mod.widget(target,op) 参数说明

属性 类型 说明
target {object|string} 需要执行模块化的对象
op {object} 参数设置
属性 类型 支持版本 说明
format {string} >= 1.0 格式化类型
= input[type = hidden]
标签控件 tags
required {boolean} >= 1.0 是否为必填项 默认 false
type {string} >= 1.0 类型。tags用参数：edit|select 通过输入来制定标签 or 通过选择来制定标签。默认为 edit
split {string} >= 1.0 输出内容分隔用字符。tags 用参数。默认为 [,] 逗号
length {number} >= 1.0 标签允许最多个数。tags 用参数。
onselect {function} >= 1.0 点击选择按钮时回调的方法。type=select 用方法

mod.widget(target,op) 返回参数说明 （target也会绑定同样方法）

属性 类型 支持版本 说明
target.add(txt) {function} 添加词条
txt {string} >= 1.0 词条名称
target.remove(txt) {function} 删除词条
txt {string} >= 1.28 词条名称
