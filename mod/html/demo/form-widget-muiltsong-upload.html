使用 HTML模板编写 不用再执行JS,填上适当的参数即可自动生成。其中 mod-xxx 为 js方法中的参数，根据 "-"来区分单词
如 titleLeft 在html中为 mod-title-left 属性。

多个歌曲上传

HTML模板写法

//多个歌曲上传
<tr>
	<th>
		<label for="multiSongUpload">多个歌曲上传</label>
	</th>
	<td>
		<input id="multiSongUpload" name="song" type="hidden" mod-required="false" mod-format="multi-song-upload">
	</td>
</tr>

javascript 写法

<!-- html -->
<table class="bs_table_s0">
				
	<tr>
		<th>
			<label for="multiSongUpload1">多个歌曲上传</label>
		</th>
		<td>
			<input id="multiSongUpload1" name="song" type="hidden" mod-required="false" mod-format="multi-song-upload">
		</td>
	</tr>
</table>
	
//js
mod.widget("#multiSongUpload1",{
	format:"multi-song-upload",
	required:true
});

mod.widget(target,op) 参数说明

属性 类型 说明
target {object|string} 需要执行模块化的对象
op {object} 参数设置
属性 类型 支持版本 说明
format {string} >= 1.0 格式化类型
= input[type = hidden]
多首歌曲上传控件 multi-song-upload
required {boolean} >= 1.0 是否为必填项 默认 false
