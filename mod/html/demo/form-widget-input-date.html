使用 HTML模板编写 不用再执行JS,填上适当的参数即可自动生成。其中 mod-xxx 为 js方法中的参数，根据 "-"来区分单词
如 titleLeft 在html中为 mod-title-left 属性。

日期

HTML模板写法

<tr>
	<th>
		<label for="date">日期</label>
	</th>
	<td>
		<input id="date" name="date" type="text" mod-required="true" mod-format="date"  mod-date-start="2014-06-15" mod-date-end="2024-06-15">
	</td>
</tr>

javascript写法

<!-- html -->
<table class="bs_table_s0">
	<tr>
		<th>
			<label for="date1">日期</label>
		</th>
		<td>
			<input id="date1" name="date" type="text">
		</td>
	</tr>
</table>		
	
//js
mod.widget("#date1",{
	required:true,
	format:"date",
	dateStart:"2014-06-15",
	dateEnd:"2024-06-15"
});

时间区间

HTML模板写法

<tr>
	<th>
		<label for="startDate">时间区间</label>
	</th>
	<td>
		<input id="startDate" name="startDate" type="text" mod-required="true" mod-format="date-interal" mod-linked="#endDate"  mod-date-start="2014-06-15" mod-date-end="2024-06-15">
		<input id="endDate" name="endDate" type="text"  mod-required="false" mod-format="date-interal" />
	</td>
</tr>

javascript写法

<!-- html -->
<table class="bs_table_s0">
	<tr>
		<th>
			<label for="startDate1">时间区间</label>
		</th>
		<td>
			<input id="startDate1" name="startDate" type="text">
			<input id="endDate1" name="endDate" type="text" />
		</td>
	</tr>
</table>
	
//js
mod.widget("#startDate1",{
	required:true,
	format:"dateInterval",
	linked:"#endDate1",
	dateStart:"2014-06-15",
	dateEnd:"2024-06-15"
});

Javascript写法

//javascript
mod.widget(target,{format:"mobile"});

mod-widget(target,op) 参数说明

属性 类型 说明
target {object|string} 需要执行模块化的对象
op {object} 参数设置
属性 类型 支持版本 说明
format {string} >= 1.0 格式化类型
日期选择器 date
日期时间选择器 date-time
日期区间选择器 date-interval
required {boolean} >= 1.0 是否为必填项 默认 false
errorText {string} >= 1.0 设置校验内容时错误的提示语
defaultText {string} >= 1.0 设置控件默认的提示语
linked {string} >= 1.0 关联
oncheck(value) {function} >= 1.0 表单校验
@param value {object} 文本框内容
@return data {object} 如果返回 true 或者 undefined 则表示成功，返回其他则当做是错误提示内容
