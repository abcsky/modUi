使用 HTML模板编写 不用再执行JS,填上适当的参数即可自动生成。其中 mod-xxx 为 js方法中的参数，根据 "-"来区分单词
如 titleLeft 在html中为 mod-title-left 属性。

单选框

HTML模板写法

<tr>
	<th>
		<label>单选框</label>
	</th>
	<td>
		<input name="rdo01" type="radio" value="0" mod-title="单选01" />
		<input name="rdo01" type="radio" value="1" mod-title="单选02" />
		<input name="rdo01" type="radio" value="2" mod-title="单选03" />
		<input name="rdo01" type="radio" value="3" mod-title="单选04" />
		<input name="rdo01" type="radio" value="4" mod-title="单选05" />
	</td>
</tr>

Javascript写法

//javascript
mod.widget(".chk1");

mod-widget(target,op) 参数说明

属性 类型 说明
target {object|string} 需要执行模块化的对象
op {object} 参数设置
属性 类型 支持版本 说明
required {boolean} >= 1.0 是否为必填项 默认 false
errorText {string} >= 1.0 设置校验内容时错误的提示语
defaultText {string} >= 1.0 设置控件默认的提示语
title {string} >= 1.0 模块名称(checkbox,radio 用参数)
img {string} >= 1.0 图片地址(checkbox,radio 用参数)
title {string} >= 1.0 模块名称(checkbox,radio 用参数)
imgWidth {string|number} >= 1.0 图片宽度，默认 100(checkbox,radio 用参数)
imgHeight {string|number} >= 1.0 图片高度，默认 100(checkbox,radio 用参数)
