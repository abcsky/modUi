HTML模板写法

使用 HTML模板编写 不用再执行JS,填上适当的参数即可自动生成。其中 mod-xxx 为 js方法中的参数，根据 "-"来区分单词
如 titleLeft 在html中为 mod-title-left 属性。

模板内容 
	<!-- html document -->
	<div class="mod-module" mod-seted="false" mod-title="模板标题" mod-title-left="左侧文字" mod-title-right="右侧文字">
	模板内容
	</div>

Javascript写法

hello kugou
	<!-- html -->
	<div id="boxTarget" style="background:gray;">hello kugou</div>		
		
	//javascript
	mod.box(target,{title:"模板标题",titleLeft:"左侧文字",titleRight:"右侧文字"})

mod.box(target,op) 参数说明

属性 类型 说明
target {object|string} 需要执行模块化的对象
op {object} 参数设置
属性 类型 支持版本 说明
title {string} >= 1.0 模块标题
titleLeft {string} >= 1.0 模块标题左侧 html内容
titleRight {string} >= 1.0 模块标题右侧 html内容
hide {boolean} >= 1.25 模块初始化时是否处于收起状态。默认为false

mod.box(target,op) 返回

属性 类型 说明
box {object} box 对象
属性 类型 支持版本 说明
box.show() {function} >= 1.41 展示模块
box.hide() {function} >= 1.41 收起模块
