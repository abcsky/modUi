模块效果

HTML模板写法

使用 HTML模板编写 不用再执行JS,填上适当的参数即可自动生成。其中 mod-xxx 为 js方法中的参数，根据 "-"来区分单词
如 titleLeft 在html中为 mod-title-left 属性。

当设置 mod-seted="true" 时将不会对其渲染。

<form class="bs_form" mod-target="#searchResultGird" mod-target-config='{action:"http://lib9.service.kugou.com/websearch/index.php", actionParam:{"cmd":100, "keyword":"123", "page":1, "pagesize":20 }, actionType:"jsonp", method:"GET", dataKey:"data.songs", totalKey:"data.total"}' >
	<table class="bs_table_s0">
		<tr>
			<th>
				<label for="keyword">关键字</label>
			</th>
			<td>
				<input id="keyword" width="200" name="keyword" type="text" mod-required="true" mod-format="" mod-autocomplete="张三,李四,黄五,赵六" >
			</td>
		</tr>
		<tr>
			<th></th>
			<td>
				<input type="button" value="提交" mod-type="submit" />
				<input type="button" value="重置" mod-type="clear" />
			</td>
		</tr>
	</table>			
</form>

Javascript写法

//javascript
mod.form(target,{
target:"#searchResultGird",
	targetConfig:{
		action:"http://lib9.service.kugou.com/websearch/index.php", 
		actionParam:{
			"cmd":100, 
			"keyword":"123", 
			"page":1, 
			"pagesize":20 
		}, 
		actionType:"jsonp", 
		method:"GET", 
		dataKey:"data.songs", 
		totalKey:"data.total"
	}
});
