组件参数

jnsCalendar(target,option)

参数说明
target {string|object}
目标对象
option {object}
参数接口

option接口内参数

参数说明
type {string}
日历类型
date:日期选择
dateTime:日期时间选择
intervalTime:日期区域选择 几号 - 几号之间
默认为:"date"
format {string}
显示格式
参数：
YYYY 年
MM 月
DD 日
hh 时
mm 分
ss 秒
默认值为:"YYYY-MM-DD"
callback {function}
选择完成后调用的方法
如果是 date|dateTime类型的，返回 对应的时间
如果是 intervalTime类型的，返回 2个参数 分别是开始时间和结束时间
默认值为function(){}
onclean {function}
执行清除操作后调用的方法
如果是 date|dateTime类型的，返回 对应的时间
如果是 intervalTime类型的，返回 2个参数 分别是开始时间和结束时间
默认值为function(){}
vDistance {number}
日历控件距离目标对象的垂直距离
默认值为：20
startDateInput {string|object}
适用于intervalTime类型控件，开始日期的input对象，读取时将直接读该对象的 value 值
默认值为：null
endDateInput {string|object}
适用于intervalTime类型控件，结束日期的input对象，读取时将直接读该对象的 value 值
默认值为：null
selectSameDate {boolean}
适用于intervalTime类型控件，允许选择同一天
默认值为：true
dateRangeStart {Date|string}
日期选择的范围-下限
默认值为：null
dateRangeEnd {Date|string}
日期选择的范围-上限
默认值为：null

jnsCalendar()提供方法：

参数说明
setOption(op) 参数设置
show(target) 日历控件显示，target为 需要控件出现的区域
hide() 日历控件隐藏
getDate() 获取控件中选中的日期
如果option.type=="intervalTime" 的话 返回 一个长度为2的数组 [date1,date2]，里面放着 选中日期的开始与结束；
否则返回选中的日期 date
init() 日历控件初始化

日历控件-通过按钮触发，然后选中之后再显示到某个元素上面

日期选择

	jnsCalendar("#btn13",{
		type:"date",
		format:"YYYY-MM-DD",
		callback:function(date){
			document.getElementById("txtbox13").value = date.getFullYear() + "-" + (date.getMonth() + 1) + date.getDate();
		},
		onclean:function(date){
			document.getElementById("txtbox13").value = "";
		}
	}).init();

范围选择

	jnsCalendar("#btn14",{
		type:"intervalTime",
		format:"YYYY-MM-DD",
		callback:function(date1,date2){
			document.getElementById("txtbox14").value = date1.getFullYear() + "-" + (date1.getMonth() + 1) + "-" + date1.getDate() + " 至 " + date2.getFullYear() + "-" + (date2.getMonth() + 1) + "-" + date2.getDate();
		},
		onclean:function(date1,date2){
			document.getElementById("txtbox14").value = "";
		}
	}).init();

日历控件-通过函数调用控件

	var myCld = jnsCalendar("",{
		type:"intervalTime",
		format:"YYYY-MM-DD",
		callback:function(date1,date2){
			document.getElementById("txtbox15").value = date1.getFullYear() + "-" + (date1.getMonth() + 1) + "-" + date1.getDate() + " 至 " + date2.getFullYear() + "-" + (date2.getMonth() + 1) + "-" + date2.getDate();
		},
		onclean:function(date1,date2){
			document.getElementById("txtbox15").value = "";
		}
	}).init();
	document.getElementById("btn15").onclick = function(){
		myCld.show(this);
	};
