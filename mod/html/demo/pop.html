通用部分

分为 一般模块(normal)，二次确认(confirm)，成功(success)，失败(error)，菊花(loading)

一般模块(normal)

页面效果

alert

Javascript写法

mod.dialog('normal','你好吗');

二次确认(confirm)

页面效果

confirm

Javascript写法

mod.dialog('confirm',{content:'你好吗',callback:function(){alert('果然很好')});

成功(success)

success

Javascript写法

mod.dialog('success','你好吗');

失败(error)

error

Javascript写法

mod.dialog('error','你好吗');

菊花(loading)

loading

Javascript写法

mod.dialog('loading','你好吗');

mod.dialog(type,op) 参数说明

属性 类型 说明
type {string} 弹窗类型
normal|success|error|loading|自定义一个名字
op {object} 参数设置
属性 类型 支持版本 说明
title {string} >= 1.0 弹窗标题
content {string} >= 1.0 弹窗内容|需要在弹窗中显示的对象(type 必须 = 自定义名字)
timeout {number} >= 1.0 弹窗显示时长
mustConfirm {boolean} >= 1.0 必须按确认才能继续操作
callback {function} >= 1.0 回调函数
cancelCallback {function} >= 1.0 取消操作的时候回调的函数(仅限于 type=confirm)
onload {function} >= 1.0 加载完成时候回调的函数
type {string} >= 1.0 弹窗类型，优先级比 type 高
width {string|number} >= 1.0 弹窗宽度。默认跟样式
height {string|number} >= 1.0 弹窗高度。默认auto
zIndex {string|number} >= 1.0 z-index 默认 100
overtime {number} >= 1.0 超过多少秒显示超时，默认跟 Config一致
delay {number} >= 1.0 延迟 多少秒显示菊花
onovertime {function} >= 1.0 请求超时后执行的方法
show {boolean} >= 1.0 是否默认显示
sizingType {string} >= 1.10 打开弹窗类型。
normal:普通
max:最大化
默认值为normal。
resize {boolean} >= 1.26 是否允许调整弹窗大小。默认跟 Config一致
minimize {boolean} >= 1.26 是否允许弹窗最小化。默认跟 Config一致
top {boolean} >= 1.29 设置弹窗是否置顶
close {boolean} >= 1.30 设置弹窗是否显示关闭按钮

mod.dialog(type,op) 返回：

属性 类型 说明
pop {object} 弹窗对象
属性 类型 支持版本 说明
pop.show() {function} >= 1.0 弹窗显示
pop.hide() {function} >= 1.0 弹窗隐藏
