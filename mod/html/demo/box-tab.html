HTML模板写法

使用 HTML模板编写 不用再执行JS,填上适当的参数即可自动生成。其中 mod-xxx 为 js方法中的参数，根据 "-"来区分单词
如 titleLeft 在html中为 mod-title-left 属性。

	<!-- html -->
	<div class="bs_tab" mod-seted="false" mod-title="一级tab模板" mod-title-left="左侧文字" mod-title-right="右侧文字" mod-onchange="document.getElementById(this.getAttribute("href").split("#").pop()).innerHTML += '{' + arguments[0] + '变！}'">
		<ul>
			<li><a href="#tabs01">标签01</a></li>
			<li><a href="#tabs02">标签02</a></li>
			<li><a href="#tabs03">标签03</a></li>
		</ul>
		<div id="tabs01">内容01</div>
		<div id="tabs02">内容02</div>
		<div id="tabs03">内容03</div>
	</div>

mod.tab(target,op) 参数说明

属性 类型 说明
target {object|string} 需要执行模块化的对象
op {object} 参数设置
属性 类型 支持版本 说明
title {string} >= 1.0 模块标题
titleLeft {string} >= 1.0 模块标题左侧 html内容
titleRight {string} >= 1.0 模块标题右侧 html内容
index {number} >= 1.0 tab初始位置
timeout {number} >= 1.0 切换停留时间
onchange(index) {function} >= 1.0 tab发生改变时回调的函数。this指向 触发的 a对象， index为当前tab的 位置。
autoPlay {boolean} >= 1.21 是否自动切换
hide {boolean} >= 1.25 模块初始化时是否处于收起状态。默认为false

mod.tab(target,op) 返回：

属性 类型 说明
tab {object} tab对象
属性 类型 支持版本 说明
tab.index {number} >= 1.0 目前在第几个tab
tab.current(index) {function} >= 1.21 切换到第几个tab
index number tab 序号
tab.next() {function} >= 1.21 切换到下一个tab
tab.prev() {function} >= 1.21 切换到上一个tab
